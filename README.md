# Belajar Basic Node JS


## Get All Notes (List Notes)

Perintah untuk mencoba get list notes seperti berikut :

```bash
node app.js list
```

Maka hasilnya akan seperti berikut :

```bash
Printing 3 note(s).
------------
Title  : Makanan
Body   : Nasi Goreng
------------
Title  : Minuman
Body   : Es Jeruk
------------
Title  : secret
Body   : Bahaya Bro
```
